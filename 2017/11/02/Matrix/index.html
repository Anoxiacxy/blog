Matrix

Posted on 2017-11-02 | In 题目 | Comments: | Views:

Description

Given an N*N matrix A, whose elements are either 0 or 1. A[i, j] means the number in the i-th row and j-th column. Initially we have A[i, j] = 0 (1 <= i, j <= N).

We can change the matrix in the following way. Given a rectangle whose upper-left corner is (x1, y1) and lower-right corner is (x2, y2), we change all the elements in the rectangle by using “not” operation (if it is a ‘0’ then change it into ‘1’ otherwise change it into ‘0’). To maintain the information of the matrix, you are asked to write a program to receive and execute two kinds of instructions.

  1. C x1 y1 x2 y2 (1 <= x1 <= x2 <= n, 1 <= y1 <= y2 <= n) changes the matrix by using the rectangle whose upper-left corner is (x1, y1) and lower-right corner is (x2, y2).
  2. Q x y (1 <= x, y <= n) querys A[x, y].

Matrix
Time Limit: 3000MS
Memory Limit: 65536K

Input

The first line of the input is an integer X (X <= 10) representing the number of test cases. The following X blocks each represents a test case.

The first line of each block contains two numbers N and T (2 <= N <= 1000, 1 <= T <= 50000) representing the size of the matrix and the number of the instructions. The following T lines each represents an instruction having the format “Q x y” or “C x1 y1 x2 y2”, which has been described above.

Output

For each querying output one line, which has an integer representing A[x, y].

There is a blank line between every two continuous test cases.

Sample Input

1
2 10
C 2 1 2 2
Q 2 2
C 2 1 2 1
Q 1 1
C 1 1 2 1
C 1 2 1 2
C 1 1 2 2
Q 1 1
C 1 1 2 1
Q 2 1

Sample Output

1
0
0
1

solution

这道题可以用二维的树状数组解决，对于0->1,1->0的转换，我们只需记录每个点的翻转次数，奇数次便是1，偶数次便是0，想办法维护这个次数，由于是平面修改，单点查询，我们可以利用差分数组的思想，将每个点的修改次数假象为二维树状数组的前缀和，那么我们就开始想办法维护这个前缀和。

如上图所示，我们希望红色区域的修改次数+1，即红色区域的前缀和+1，那么可以这么做：

这样，我们先对[x,n][y,n]的前缀和整体+1，再对[x,n][y1+1,n]和[x1+1,n][y,n]整体-1，最后将[x1+1,y1+1]，那么红色区域整体前缀和+1，蓝色区域+1-1，绿色区域+2-2，所以恰好搞定了平面修改的问题，这时我们的前缀和便是单点修改次数，查询只需sum[x,y]即可
